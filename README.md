# 微信公众号排版工具

一款专为微信公众号创作者打造的桌面端排版工具，提供丰富的样式模板、AI 写作辅助和一键导出功能，让排版变得简单高效。

## 下载安装

前往 [Releases](https://github.com/maiqingbu/we/releases) 页面下载对应系统的安装包：

- **macOS**：下载 `.dmg` 文件，打开后将应用拖入"应用程序"文件夹。首次打开时，**右键点击应用 → 选择"打开"**（因为未签名，直接双击会被系统拦截）
- **Windows**：下载 `-setup.exe` 文件，双击运行安装程序

## 界面介绍

程序分为三个区域：

- **左侧边栏**：文章列表，切换和管理多篇文章
- **中间编辑区**：文章编辑区域，顶部为工具栏
- **右侧预览区**：实时预览文章在公众号中的最终效果

## 创建与编辑文章

### 新建文章
点击左侧边栏底部的"+"按钮创建新文章。

### 基本排版
使用顶部工具栏进行排版：
- **标题**：选择 H1-H4 级别标题
- **文字样式**：粗体、斜体、下划线、删除线、行内代码
- **文字颜色**：设置文字颜色和背景高亮色
- **字体**：调整字体大小和字体
- **段落**：行高、段间距、首行缩进、文字对齐方式
- **列表**：有序列表、无序列表、任务清单
- **其他**：引用块、代码块、表格、分割线

### 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl/⌘ + B` | 粗体 |
| `Ctrl/⌘ + I` | 斜体 |
| `Ctrl/⌘ + U` | 下划线 |
| `Ctrl/⌘ + Z` | 撤销 |
| `Ctrl/⌘ + Shift + Z` | 重做 |
| `Ctrl/⌘ + F` | 查找 |
| `Ctrl/⌘ + H` | 查找替换 |
| `Ctrl/⌘ + Shift + E` | 复制到公众号 |
| `/` | 打开快捷插入菜单 |

### 斜杠命令
在编辑器中输入 `/` 可以快速插入：标题、列表、引用、代码块、图片、链接、表格等。

## 主题样式

点击右侧预览区顶部的主题选择器切换文章样式。

**内置 9 套主题**：默认、优雅、科技、养生、本草纲号、食验室、蓝图、折叠空间。

**自定义主题**：在主题选择器中点击"自定义"，可以基于现有主题修改 CSS 样式，创建专属排版风格。

## 素材库

点击顶部工具栏的素材按钮打开素材库，包含以下分类：

### 分割线
47 种分割线样式，分为极简、图案、渐变、装饰四大类。点击即可插入到文章中。

### 模板
18 种实用模板，点击插入后可直接编辑内容：
- 信息盒（提示/注意/重点）
- 引用卡、高亮重点
- 问答卡片、优劣对比
- 步骤引导、编号列表
- 数据统计卡、关键要点
- 警告提示、用户评价
- CTA 按钮、二维码卡
- 作者信息卡、关注引导
- 目录、文章封底

### 节日素材
春节、中秋、圣诞、七夕、国庆等节日专属素材，临近节日时会自动推荐。

### SVG 素材
80 种矢量素材，包含装饰图案、常用图标、徽章标签。

### 自定义素材
选中编辑器中的内容，右键 → "保存为素材"，即可将常用样式保存下来反复使用。支持分组管理、导入和导出。

## 图片处理

### 插入图片
- 点击工具栏的图片按钮插入本地图片
- 支持直接粘贴图片

### 图片编辑
双击编辑器中的图片打开图片编辑器，支持：
- 裁剪、旋转
- 滤镜效果
- 亮度、对比度、饱和度调节
- 添加文字、贴纸、水印
- 图层管理

### 图床上传
在设置中配置图床（支持 SM.MS、GitHub、自定义接口），插入的图片会自动上传，避免文章体积过大。

## AI 写作助手

### 配置
首次使用需要在设置中配置 AI 服务：
- 点击右上角设置按钮
- 选择 AI 平台（DeepSeek / 通义千问 / 豆包）
- 输入 API Key 并保存

### 使用方式

**选中文本使用**：选中一段文字后，点击出现的 AI 按钮，可选择：
- 润色：优化表达，使文字更流畅
- 缩写：压缩到原文 60-70%
- 扩写：扩展到原文 1.5-2 倍
- 改风格：专业、口语、幽默、严肃、文学五种风格
- 翻译：翻译为英文、日文、韩文
- 自定义：输入任意指令让 AI 执行

**全文功能**：在编辑器空白处使用斜杠命令 `/`：
- 生成标题：AI 生成 5 种风格的候选标题
- 生成摘要：自动生成 100-150 字的文章摘要

### 全文校对
底部状态栏显示错别字和敏感词数量，点击可查看详细列表并跳转到对应位置。

## 导入文章

点击顶部菜单的导入按钮，支持以下格式：
- **Word** (.docx)：自动转换为富文本
- **Markdown** (.md)：保留格式结构
- **PDF**：提取文字内容
- **网页链接**：输入 URL 自动抓取正文内容

## 导出文章

### 复制到公众号
点击右上角"复制到公众号"按钮，打开微信公众号后台编辑器，直接粘贴即可。样式会自动保留。

### 导出为文件
点击"导出"按钮，支持以下格式：
- **长图** (PNG)：适合发微博、朋友圈、小红书，可选不同宽度
- **PDF**：适合打印或归档
- **HTML**：完整网页文件，保留所有样式
- **Markdown**：纯文本格式，方便二次编辑

## 文章管理

### 多文章管理
左侧边栏显示所有文章，点击切换。支持新建和删除文章。

### 版本历史
编辑器会自动保存文章快照（最多 30 个版本），点击工具栏的版本历史按钮可查看和恢复历史版本。

### 样式收藏
将常用的样式配置保存为收藏，方便在不同文章中复用。

## 底部状态栏

底部状态栏实时显示：
- 文章字数、段落数、图片数、链接数
- 预估阅读时间
- 错别字数量（点击查看详情）
- 敏感词数量（点击查看详情）
